# C++ Functions Practice

This program demonstrates the use of **functions in C++**. It includes multiple functions that perform different mathematical operations based on user input.

## Features

* Calculates the **product of two numbers**
* Checks whether a number is **even or odd**
* Calculates the **factorial of a number**
* Computes the **binomial coefficient**

## Concepts Used

* Functions in C++
* Function parameters
* Loops
* Conditional statements
* Basic mathematical calculations

## How It Works

The program asks the user to enter values and then calls different functions to perform the required calculations. Each function executes a specific task and displays the result.

## Language Used

C++
